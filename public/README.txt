Place your LUXWELD logo here as:

    luxweld-velux-trusted.svg

When this file exists, the app uses it automatically in the header and on the
login screen. Until then, a styled "LUXWELD" wordmark is shown as a fallback.

Files in this folder are served as static assets at the site root, e.g.
public/luxweld-velux-trusted.svg  ->  /luxweld-velux-trusted.svg
