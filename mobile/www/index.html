Set your deployed app URL

Set your deployed app URL before building the mobile app.

Edit mobile/www/app-config.js and set:

window.LUXWELD_APP_URL = "https://warranty.luxweld.com.au";

…or run LUXWELD_APP_URL=https://… npm run set-app-url in the project root, then rebuild.

localhost will NOT work from a phone — use a LAN IP (http://192.168.x.x:3000) or an HTTPS tunnel (ngrok / cloudflared). Production builds should use your real HTTPS domain.
